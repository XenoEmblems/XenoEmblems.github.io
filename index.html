Safari Zone

Summary


A full stack Ruby on Rails application that is built as the final project of the General Assembly Web Development Immersive Course. The app is supposed to recreate the Pokemon Safari Zone minigame in the mainstream Pokemon Games. When an account is created then Pokemon are rendered on the front page then you can capture and learn about the Pokemon you find.


The database was filled using Pokéapi which is a free Pokemon API. In V1 the missing catchrate in the database was inserted using JSON. Currently the API updated to V2 and now has the correct catchrates.

Rozay n ur Resume

Summary


This was a group project built with William Barbee and Nicholas Buttersworth for the General Assembly Web Development Immersive.


This is a full stack web application using JavaScript accessing the Indeed, Glassdoor, TheMuse, and Craigslist APIs to gather web development job openings and render the job information when they are clicked on. Information of the rating of the companies and how jobs are rated and their average salary is collected from the Glassdoor API

Alcove

Summary


This is a custom Wordpress website which was made to showcase the upcoming independent flannelcore band Alcove. It was built by making a child theme of TA-Music which had great features but had to be debugged and edited for more recent versions of Wordpress.


The website was built to showcase their next concerts, biographies, and history of the band. Features include an online ticketing system and widgets showing their Facebook and Soundcloud feed.
